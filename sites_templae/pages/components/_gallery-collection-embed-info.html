Intro to Embedding

Use the below form to create a link that you can copy/paste into your own website (embed). When a user clicks the link, a viewer will open that shows the artwork in this collection.

  • You can display the link as a plain link, or an image.
  • If a title image has been set, this will be shown. Otherwise, the first artwork in the collection will be used.
  • By default, images are 200x200px. If you enter "large" in Container Class, it will be 500x500px.
  • You can further style the container containing the link or image using your own CSS.
  • You can choose set the size of the image by adding CSS like "width: 300px;height:400px".
  • For plain links, the font and color of your own website will be applied.
  • The title below will be used for the link text, or shown on the image when the user hovers on the image.

After adding a new embed, use the preview link to get the code you need to copy to your website
